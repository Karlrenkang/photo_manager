# 照片去重工具 V1

一个简单高效的命令行照片去重工具，通过文件大小和 MD5 哈希值检测并移除重复图片。

## 功能特点

- ✅ 支持多种图片格式：JPG、JPEG、PNG、WebP、HEIC
- ✅ 递归扫描子文件夹
- ✅ 两步去重：先按文件大小分组，再计算 MD5 确认
- ✅ 自动保留文件名最小的图片，移动重复文件到 duplicates 文件夹
- ✅ 纯命令行界面，无需 GUI
- ✅ 单线程执行，简单可靠
- ✅ 完整的中文注释


## 使用方法

```bash
python main.py "目标文件夹路径"
```

### 示例

```bash
# 扫描 D:/photos 文件夹
python main.py "D:/photos"

# 扫描当前目录下的 photos 文件夹
python main.py "./photos"
```

## 工作流程

1. **扫描图片**：递归扫描指定文件夹，找出所有支持的图片文件
2. **按大小分组**：将文件大小相同的文件分为一组（大小不同的文件肯定不重复）
3. **计算 MD5**：对大小相同的文件计算 MD5 哈希值
4. **识别重复**：MD5 相同的文件被认为是重复文件
5. **保留最优**：每组重复文件中，保留文件名最小的那张
6. **移动重复**：将其余重复文件移动到 `duplicates` 文件夹

## 注意事项

- ⚠️ 重复文件会被移动到 `duplicates` 文件夹，**不会直接删除**
- ⚠️ 建议先备份重要照片再使用
- ⚠️ 如果目标文件夹中已有同名文件，会自动添加序号避免覆盖
- ⚠️ 仅支持 JPG、JPEG、PNG、WebP、HEIC 格式

## 技术细节

- **去重策略**：文件大小 + MD5 哈希双重验证
- **保留规则**：文件名按字典序排序，保留第一个
- **文件格式**：通过扩展名识别，不区分大小写
- **性能优化**：先按大小分组，只对可能重复的文件计算 MD5
